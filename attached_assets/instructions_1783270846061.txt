**Role:** Senior UI/UX Designer & Frontend Specialist for Fintech.
**Platform:** React Native (Expo) / Flutter compatible component library.
**Design System Name:** Red Velvet Black.
**Brand:** TrustPoint Bank — "Security Meets Modernity."
**Core Palette:**
- **Primary Red:** `#E63946` (Vibrant, trust-energizing red).
- **Primary Black (Dark Mode):** `#0A0A0A` (Deep void black, never pure `#000`).
- **Secondary Dark Gray:** `#1A1A1A` (Cards, surfaces).
- **Accent Gold/White:** `#F1FAEE` (Text on dark, subtle highlights).
- **Light Mode Background:** `#F8F9FA` (Soft off-white).
- **Light Mode Surface:** `#FFFFFF` (Pure white cards).
- **Soft Shadows:** Use `shadow-color: #E63946` with 10% opacity for a "red glow" effect on elevated cards; standard black shadows with 5% opacity for light mode.

**Global Design Rules (CRITICAL):**
1.  **Soft & Smooth UI:** Every interactive element must have a `border-radius` of 16px (cards), 12px (buttons), or 50px (input fields). No sharp corners.
2.  **Micro-interactions:** All buttons must have a `scale: 0.97` on press. Transitions must be `spring-based` (damping: 15, stiffness: 170).
3.  **Depth:** Use a 3-layer shadow system. Layer 1: subtle spread. Layer 2: soft blur. Layer 3: a colored glow (red in dark mode, faint grey in light mode).
4.  **Typography:** Use only Inter/SF Pro Rounded. Headings: Semi-bold (600). Body: Regular (400). Captions: Medium (500). Font smoothing: `antialiased`.
5.  **Mode Toggle:** A sleek slider switch in settings instantly swaps the entire `Red Velvet Black` theme to the `Soft Crimson Light` theme.
